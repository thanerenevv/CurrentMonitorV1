Web Flasher

CURRENT MONITOR // OVER USB

Connect your ESP32-C3 via USB
Requires Chrome or Edge on desktop — USB cable directly to the ESP32-C3.

On first boot the device creates a WiFi AP named PowerMonitor-Setup (password: configure). Connect to it from your phone or laptop to set your network credentials.

To reset WiFi settings after flashing, visit http://<device-ip>/reset.

Default mains voltage is 230V AC. If you're on 120V, edit config.h and build from source.
